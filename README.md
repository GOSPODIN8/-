# 📖 Бот «33 жёсткие истины» — Эдриан Трейп

Telegram-бот для продажи PDF-книги через **Telegram Stars ⭐️**

---

## 🚀 Запуск через Railway (бесплатно, 5 минут)

### Шаг 1 — Создай аккаунты

| Сервис | Зачем | Ссылка |
|--------|-------|--------|
| GitHub | Хранить код | github.com |
| Railway | Запустить бота | railway.app |

---

### Шаг 2 — Создай бота в Telegram

1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя: например `Книга Эдриана Трейпа`
4. Придумай username: например `adriyan_book_bot`
5. Скопируй **токен** — он выглядит так:
   ```
   1234567890:ABCDefGHIjklMNOpqrSTUvwxYZ
   ```

---

### Шаг 3 — Залей на GitHub

1. Зайди на **github.com** → нажми **New repository**
2. Назови репозиторий: `adriyan-book-bot`
3. Нажми **Create repository**
4. Загрузи все файлы из этой папки:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `runtime.txt`
   - `33_istiny_ru.pdf`
   - `33_hakikat_tg.pdf`

> Как загрузить файлы: на странице репозитория нажми
> **Add file → Upload files** → перетащи все файлы → **Commit changes**

---

### Шаг 4 — Запусти на Railway

1. Зайди на **railway.app**
2. Нажми **Start a New Project**
3. Выбери **Deploy from GitHub repo**
4. Выбери свой репозиторий `adriyan-book-bot`
5. Railway начнёт деплой — подожди 1-2 минуты

**Добавь токен бота:**
1. В Railway открой свой проект
2. Нажми на сервис → вкладка **Variables**
3. Нажми **New Variable**
4. Имя: `BOT_TOKEN`
5. Значение: вставь свой токен от @BotFather
6. Нажми **Add**

Railway автоматически перезапустит бота с токеном.

---

### Шаг 5 — Проверь

Открой Telegram → найди своего бота → напиши `/start`

Должно появиться меню с кнопками! 🎉

---

## 💰 Как работает оплата

```
Покупатель → /start → выбирает версию
    → Telegram открывает окно оплаты Stars
    → Покупатель платит ⭐️
    → PDF приходит автоматически
    → Stars поступают тебе
```

**Цены:**
- 🇷🇺 Русская версия → **50 ⭐️**
- 🇹🇯 Таджикская версия → **50 ⭐️**
- 📦 Обе версии → **80 ⭐️**

---

## 💸 Вывод Stars

1. Открой @BotFather в Telegram
2. Выбери своего бота
3. **Bot Settings → Revenue**
4. Там кнопка вывода средств

---

## ✏️ Изменить цены

Открой `bot.py`, найди строки:

```python
PRICE_RU   = 50
PRICE_TG   = 50
PRICE_BOTH = 80
```

Измени цифры → сохрани → залей обновлённый файл на GitHub.
Railway обновит бота автоматически.
